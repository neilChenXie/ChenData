### 创建用户

```bash
# use root
su
useradd -m hadoop -s /bin/bash # usrname=hadoop
```

### 设置密码

```bash
passwd hadoop
```

### 将创建的用户加入root group

```bash
visudo
```
用tab分割

root    ALL=(ALL) ALL
hadoop  ALL=(ALL) ALL
